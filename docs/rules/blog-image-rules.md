# Blog Image Rules

## Goal

Images should support the blog topic, improve trust, and help homeowners understand the work.

Images should look realistic and connected to EPF Pro Services work.

## Image requirement for new blogs

Every new blog should include a realistic featured image or a detailed featured image prompt.

If image generation is available and the user has not said to skip images, generate or add one realistic featured image for the blog.

If image generation is unavailable, include a complete image plan and featured image prompt so the asset can be created later.

## When images are requested

If the user asks to generate images, create image prompts, add images, create a featured image, or plan visuals for a blog, always create an image plan first.

The image plan should include:

1. Featured image
2. 2-4 supporting images
3. Optional before/after image if relevant
4. Optional process image if relevant
5. File name suggestions
6. Alt text
7. Caption suggestions when useful

For full service-support blogs, include useful image coverage when available:

- One close-up or problem image
- One wide room or context image
- One process image
- One finished or paint-ready result image

Never repeat the same alt text.

Never repeat generic captions.

Do not claim a photo is from an EPF job unless that is confirmed.

If using generated, stock, placeholder, or generic site images, describe what is visible without calling it a real EPF project photo.

## Image types by blog

### Cost guide image

Use images such as:

- Clean finished room with smooth ceiling
- Contractor reviewing ceiling or estimate
- Protected room before ceiling work
- Simple process/cost visual if the design supports it

### Process guide image

Use images such as:

- Room protection
- Popcorn removal in progress
- Skim coating stage
- Sanding stage with dust control
- Priming/painting stage
- Finished smooth ceiling

### Drywall repair image

- Finished painted wall

### Interior painting image

- Clean cut lines

### Wallpaper removal image

Use images such as:

- Wallpaper removal in progress
- Wall after wallpaper removal
- Wall repair/prep before paint
- Finished painted wall

### Baseboard image

Use images such as:

- Baseboard installation in progress
- Caulking/filling nail holes

## Preferred image style

Use:

- Realistic professional home renovation photography
- Ontario/GTA residential interior
- Natural daylight
- Clean work area
- Practical contractor workflow
- Realistic materials and surfaces
- Wide-angle editorial style when useful
- No exaggerated luxury design
- No fake perfect surfaces if the image is a process image

Avoid:

- Text inside the image
- Fake logos
- Fake watermarks
- Fake brand labels
- Unsafe work
- Messy dangerous jobsite
- People's faces unless requested
- Unrealistic luxury mansion look
- Cartoon style unless user asks
- Overly glossy AI look

## Image prompt template

Use this prompt structure:

"Realistic professional home renovation photo of [subject/work stage] in a [room type] in an Ontario/GTA home. Show [specific details]. Clean protected work area, natural daylight, realistic contractor workflow, high detail, wide-angle editorial style. Avoid text, logos, fake watermarks, unsafe work, exaggerated luxury design, and unrealistic perfect surfaces."

## Featured image rules

The featured image should:

- Clearly match the blog topic
- Work in a wide format
- Look trustworthy
- Not be too busy
- Not include text baked into the image
- Not include fake logos or fake watermarks
- Have a descriptive file name
- Have descriptive alt text

## File naming

Use lowercase hyphenated file names.

Examples:

- popcorn-ceiling-removal-cost-mississauga-featured.webp
- painted-popcorn-ceiling-skim-coat-process.webp
- drywall-patch-before-painting.webp
- smooth-ceiling-after-popcorn-removal.webp
- interior-painting-wall-preparation.webp

## Alt text

Alt text must describe the actual image.

Good:

"Protected living room before popcorn ceiling removal in a Mississauga home."

Good:

"Skim coating stage after painted popcorn ceiling removal."

Good:

"Finished smooth ceiling after primer and flat ceiling paint."

Bad:

"Best popcorn ceiling removal Mississauga cheap popcorn ceiling removal popcorn ceiling company."

Bad:

"Popcorn ceiling removal popcorn ceiling removal popcorn ceiling removal."

## Captions

Use captions when they help explain the work.

Examples:

"Room protection is usually the first step before popcorn ceiling removal begins."

"Skim coating helps flatten the ceiling after the texture is removed."

"Flat ceiling paint helps hide minor reflection and keeps the ceiling looking smooth."

## Image SEO

When adding images, include:

- Descriptive file name
- Descriptive alt text
- Relevant caption when useful
- Optimized image format, preferably WebP if supported
- Width/height attributes if the project supports them
- Lazy loading for non-hero images
- Priority loading only for the main hero/featured image if the project supports it
- Compressed images where possible
- Unique alt text for every image
