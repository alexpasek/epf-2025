# Internal Linking Rules

## Purpose

Blogs should help users find the right service page and pass topical relevance to important money pages.

Internal links should be useful, natural, and contextually relevant.

Every blog should also strengthen a clear topical cluster instead of standing alone.

## Topical cluster linking

Before adding links, identify the blog's topical cluster.

Then connect the article to:

- The main pillar page or money page for that topic
- One relevant city/service page if the article is local
- One to three closely related blog articles when available
- One adjacent service page if it helps the reader solve the next problem
- The contact/quote page near the CTA when useful

Examples:

- A popcorn ceiling cost blog should support the popcorn ceiling removal cluster.
- A painted popcorn ceiling comparison blog should connect to popcorn removal, smooth ceiling refinishing, and related ceiling finish guides.
- A drywall patch problem blog should support the drywall repair cluster and may connect to interior painting prep where natural.
- A wallpaper removal blog should connect to wallpaper removal, wall repair/prep, and interior painting when relevant.

Avoid orphan posts.

Avoid linking to unrelated service pages just to add more links.

## Authority flow rules

Use internal links to pass topical authority clearly.

Every full blog should:

- Link to the main pillar/money page in the first third of the article.
- Link to the most relevant city page when the blog has local intent.
- Link to 2-4 related articles inside the same topical cluster when available.
- Link to one adjacent service page only when it helps the reader take the next step.
- Link to the quote/contact page in the CTA section.
- Use varied descriptive anchors instead of repeating the same exact anchor every time.
- Keep the first service link close to the reader's problem, not forced into the first sentence.
- Avoid footer-only authority links. Important links should appear naturally in the body copy.

For new cluster posts, check whether older related posts should link back to the new post. Add backlinks only when useful and requested, or when the current task includes cluster strengthening.

## Required links

Each blog should usually include:

- Main service hub link
- Relevant city/service page link if local
- Related service page links where useful
- Related blog article links where useful
- Contact/quote page link near CTA if available

## Main money pages to support

Support these types of pages:

- Popcorn ceiling removal main page
- Popcorn ceiling removal city pages
- Drywall repair main page
- Drywall repair city pages
- Drywall installation page
- Interior painting page
- Wallpaper removal page
- Baseboard installation page
- Contact/quote page

## Anchor text

Use descriptive anchor text.

Good:

- professional popcorn ceiling removal
- popcorn ceiling removal in Mississauga
- smooth ceiling refinishing
- drywall repair services
- interior painting services
- Level 5 ceiling finishing
- popcorn ceiling removal cost guide
- request a popcorn ceiling removal quote

Bad:

- click here
- this page
- read more
- learn more
- link
- visit

## Link placement

Add links naturally:

- One early contextual service link in the intro or first third of the article
- One city/service page link where location is discussed
- Related article links near the middle or bottom
- CTA/contact link near the end

## Link limits

Do not overload the page.

Most blogs should have 4-8 internal links depending on article length.

Short articles may need fewer.

Long guides may need more if useful.

For 2,200+ word blogs, 6-10 useful internal links is acceptable when they are natural and cluster-relevant.

## Authority flow

For a blog supporting a money page:

- Link to the main service page using a natural service anchor.
- Link to the city page using a city/service anchor if local.
- Link to 1-3 closely related blogs.
- Link within the same topical cluster before adding broader adjacent-topic links.
- Link to contact/quote page near the CTA.
- Do not link to unrelated pages just to add more links.

## Blog-to-blog linking

When useful, link supporting blogs together.

Examples:

A popcorn ceiling cost blog can link to:

- Painted popcorn ceiling removal guide
- Popcorn ceiling removal process guide
- Smooth ceiling finishing guide
- Popcorn ceiling removal FAQ

A drywall repair blog can link to:

- Drywall patch repair guide
- Why drywall patches show after painting
- Drywall repair cost guide
- Interior painting preparation guide

## Local linking

For city blogs:

- Link to the matching city/service page.
- Link to the main service page.
- Link to nearby city pages only if useful and natural.
- Do not create long lists of cities inside every article.
- Do not repeat the same city links unnaturally.

## Footer/related section

Add a "Related guides" or "Related local pages" section when useful.

Keep it short and relevant.

For important full blogs, include a related local pages/service links section when the article supports local service intent. This section should connect the reader to the matching service page, city page, related cluster guides, calculator when relevant, and quote/contact page.

Example:

Related guides:
- Popcorn ceiling removal cost guide
- Painted popcorn ceiling removal
- Smooth ceiling refinishing
- Drywall repair before painting

## Final internal-linking check

Before finishing, confirm:

- Topical cluster identified
- Pillar page or money page linked
- Pillar/money page linked in the first third where natural
- Main service page linked
- City page linked if local
- Related blog/service links added
- 2-4 same-cluster blog links added where available
- Quote/contact link added near the CTA where useful
- Cluster links are natural and helpful
- Anchor text is descriptive
- No "click here" anchor text
- No irrelevant links
- No excessive linking
